## Install

```sh
npm i plix -g
```

## Help

```sh
plix help
```

## Create a plix blog

```sh
plix new my-fabulous-blog
cd my-fabulous-blog
```

## Create a plix page

```sh
plix page my-first-page
```

## Build

```sh
plix build
```

## Deploy

```sh
plix deploy
```
